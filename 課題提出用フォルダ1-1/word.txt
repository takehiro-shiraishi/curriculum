＜ITについて>
ITとは、information technologyの略でインターネットなどの通信とコンピュータとを駆使する情報技術です。
＜OSについて＞
OSとはOperation Systemの略で、アプリやデバイスを動作させるための基本となるソフトウェアのことです。具体的には、キーボードやマウス、タッチパッドなどのデバイスから入力した情報をアプリケーションに伝え、またソフトウェアとハードウェアの連携を司る中枢的な役割を果たします。パソコンやスマートフォンにはゲームやワープロ、表計算などの様々な仕事をするアプリケーションがありますが、それらはOS毎に開発されるのが通常です。
・OSの種類
WindowsOS マイクロソフトが開発、販売するOS
Mac OS　アップルが開発販売するOS
UNIX ベル研修所が発表したもの主に企業の中のサーバー、ワークステーション上で稼働している
Linux オープンソース（無料で使えて、中身もいじれる）のOSで主に企業のサーバーで使われることが圧倒的に多い

＜言語・環境＞
HTML 　Hyper Text Markup Languageの略でウェブページを作成するために開発されたマークアップ言語です。

CSS  Cascading Style Sheetsの略でウェブの文字の色や書体などのスタイルを指定するためのスタイルシート言語の一つです。

MySQL  世界中の多くの企業が使用しているデータベース管理システムで大容量のデータを高速に行え、レンタルサーバーや検索エンジンでも使用されている。

PHP  動的にwebページを生成することができるサーバーサイド（サーバ側で動くシステム）スクリプト言語です。webアプリケーションの開発によく使われる。

JavaScript  動的なwebページを作成できるプログラミング言語で、通常はブラウザ上で表示されます。地図やグラフィックアニメーションなどの表示をすることができます。
jQuery  JavaScriptコードをより容易に記述できる様にするために設計されたライブラリ。ブラウザ毎の差異をなくすことができる。

Apache  世界中で使用されているwebサーバーソフトウェア(HTTPサーバ)で、無償で誰でも利用することができる。また、セキュリティー対策が随時更新されているので安心。

Bootstrap  web制作の作業を楽にする目的で作られた、webサイト構築用のCSSのひな形セットの名前です。実態はいくつかのCSSファイルとJavaScriptファイルです。

＜エンジニアなど＞
SE  顧客の要求を実現する要求分析・要件定義、顧客のニーズを実現するシシステムの設計を行う基本設計、実装予定の機能に対しどの様な技術や仕組みで実現するのかを設計する詳細設計、システムがきちんと設計どうりの動作をするかテストをするテストなどを担当します。

PG  SEが作った仕様書を基にシステムが作動する様にプログラムを書く仕事。

Webデザイナー　　企業や個人などのクライアントから依頼されたwebデザイン（HTML,CSS,JavaScript等)を使用し担当する仕事です

＜ユーザーインターフェイス＞
ユーザーとコンピュータ間での情報のやりとりの方法、仲立ちをするという意味です。

CUI  Character、文字を使ったもの

GUI  Graphical、グラフィックや図表を使ったもの

＜ファイルとディレクトリ＞
ファイル　　データはファイルと呼ばれるもので管理される。また名前をつけることができ、所有権やアクセス権という属性を持ちます。

ディレクトリ　　階層構造（ツリー構造）を実現するための特別なファイルのことをディレクトリと呼びます。

ルートディレクトリ　　階層型ファイル構造における最上層のディレクトリのこと

カレントディレクトリ 　ユーザーが現在作業を行なっているディレクトリのこと

パス　　パソコン内住所の様なもの、ホームページのアドレス表記などにも使われる

絶対パス　　最上位の階層を起点にパスをフルで表記すること

相対パス　　カレントフォルダを起点として目的のファイルがどこにあるかを記述する方法

＜各種プロトコル＞
HTTP　　Hyper Text Transfer Protocolの略ホームページのデータ（文字、画像、動画）をサーバとクライアントの間で通信するときの通信規約（プロトコル）がHTTP

HTTPS　　通信内容が暗号化されたHTTP

FTP　　特定ホスト間でファイルを転送するために作られたプロトコル

SSL  インターネット上におけるウェブブラウザとウェブサーバ間でのデータの通信を暗号化し、送受信させる仕組みのことです。

SSH  暗号化されたログインシステムのこと

ポート、ポート番号　　ポートはネットワークとパソコンの間にあるドアの様なもの。コンピュータが通信を行うために通信先のアプリケーションを特定する為の番号のこと。ポート番号はポートに割り振られている識別用の番号のこと。
・HTTP  TCP80
・FTP  TCP20(データ),TCP21(制御)
・SSH  TCP22  

ファイヤウォール　　通信させてはいけない通信を阻害するシステム

TCP  コネクション型通信（通信開始前に相手との間で仮想的な通信路を確保する）を行います。
UDP  コネクションレス型通信（通信開始前に相手のと間で、事前のやり取りなし）を行います。

＜IPアドレス＞
IPアドレス　　パソコンやスマートフォンなどのネットワーク上の機器を識別する為に割り当てられていて、インターネット上での住所の様な役割を担っている

グローバルIPアドレス　　インターネットに接続するタイプのIPアドレスです。

プライベートIPアドレス　　インターネットに接続しないタイプのIPアドレスです。

＜ドメイン＞
ドメイン　　インターネット上に定められた部分領域のこと、ネット上の住所

DNS　　ドメインとIPアドレスを対応付けて管理するシステムのこと

VPN   インターネットに接続している利用者の間に仮想的なトンネルを構築して、プライベートなネットワークを拡張する技術です。

ブリッジ　　ネットワーク中のデータを見て、行先を振り分けてくれる機器のこと

プロキシサーバ　　ホームページを見るときに使うソフト（webブラウザ）の身代わり担ってホームページにアクセスしてくれるコンピュータ（サーバ）のこと。

＜電子メール＞
SMTP  メールを相手のメールサーバまで届ける仕組みのこと

POP  メールソフトが自動的にメールサーバにアクセスしてメールをダウンロードしている仕組みのこと

IMAP  サーバにあるメールをパソコン等の端末にダウンロードせず、サーバ上でメールを管理する仕組みのこと

＜セッションとクッキー＞
クッキー　　ユーザーのコンピュータに保存される小さなファイルのことです。

セッション　一時的にサーバ上のディレクトリに保存されるデータのこと

キャッシュ　　よく使うデータへのアクセスを早くするために、より高速な記憶装置に一時的に保存する仕組みのことを指す

＜データベースについて＞
チューニング　　性能の悪いSQLを発見し、その情報過程を改善することを指す。目的の処理結果を変えずにその過程を最適化すること。

スロークエリ　　データベースにおいて時間のかかっているSQLのことをいう。

インデックス　　テーブルの情報を探すときに検索の対象としてよく使用するカラムの値だけを取り出して検索しやすい様にしておいたものです。

＜テストについて＞
フロー　　制御フローテストは命令文、分岐した経緯、条件に着目し、これらが全て実行されるかを確認します。データフローテストはデータや変数が定義、使用、消滅の順に正しく処理されているかを確認するテストです。

単体テスト　　プログラムを構成する比較的小さな単位がここの機能を正しく果たしているかを検証するテスト

結合テスト　　手続きや関数といった個々の機能を結合させて、上手く連携、動作しているかを確認するテストのことである。

＜チケット管理ツール＞
チケット管理ツール　　実施するべき作業や課題をチケットとして管理する方法。発生日や作業内容、優先度、担当者や期日を登録して管理します。

タスク　　任せられた仕事、作業のこと

＜バージョン管理＞
Git　　分散型バージョン管理システムの一つで、元々Linuxのソースコードを効果的に管理するために開発されました。gitではファイルの状態を好きなときに更新履歴として保存しておくことができます。そのため、一度編集したファイルを過去の状態に戻したり、編集箇所の差分を表示したりすることができます。

SourceTree　　Atlassianが無料で提供しているGitを用いたバージョン管理ツールです。GUIで表現されているため視覚的にわかりやすいことが特徴。

SVN　　1つの中央リポジトリに複数の開発者がアクセスするアーキテクチャを採用している。コミット毎に新しいリビジョン番号が振られるためバージョンの管理が直感的に行えるほか対応するクライアントが多い

コンフリクト　　二つの分岐で不整合が起きてしまい、履歴としてその状態から進ませることができなくなること。衝突ともいう

デグレ　　システム開発におけるプログラム修正やインフラの設定変更等により、それまで正常に動作していた機能が動かなくなるという、品質低下のトラブル事象を指す。

マージツール　　コンフリクトしたりした際にそれを比較してマージする際に、手戻りが発生するなどして作業時間が大幅にかかってしまうなどのチーム開発における非生産的な作業を効率化するもの。
